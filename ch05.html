R 的資料型態

  • R 包含三種基本資料型態 (class)：
    1. 數值 (numeric)： x <- c(174, 158, 160, 168, 173)
    2. 字串 (character)： y <- c("林志玲", "蔡依林", "楊丞琳", "天心", "隋棠")
    3. 邏輯 (logical)： z <- c(TRUE, FALSE, FALSE, FALSE, TRUE)
  • 利用class函數判斷物件型態
  • character 內容以雙引號 (" ") 或單引號 (' ') 註記
  • logical 內容以TRUE / FALSE 註記，亦可簡化為 T / F
class(x); class(y); class(z)
[1] "numeric"
[1] "character"
[1] "logical"

資料型態的轉換

  • 向量物件一次只能接受一種型態
  • 在向量中給定多種型態，將會依序自動轉換：字串 > 數值 > 邏輯。
# 布林值 TRUE 被轉換成1，FALSE被轉換成0
c(174, 52, TRUE) 
[1] 174  52   1
c(1.1, 2.4, TRUE, FALSE)
[1] 1.1 2.4 1.0 0.0


c(174, 52, "林志玲") # 數值被轉換成字串
[1] "174"    "52"     "林志玲"
# 所有元素都被轉換成字串
c("林志玲", 174, 52, TRUE)
[1] "林志玲" "174"    "52"     "TRUE"

資料型態的轉換2

  • 利用以下函數自行轉換向量型態： as.character, as.numeric, as.logical。
# 字串轉數字
a1 <- c("89", "91", "102")
as.numeric(a1)
[1]  89  91 102
# 布林轉數字
a2 <- c(TRUE, TRUE, FALSE)
as.numeric(a2)
[1] 1 1 0


# 數字轉布林
a3 <- c(-2, -1, 0, 1, 2) # 只有0會被轉成FALSE
as.logical(a3)
[1]  TRUE  TRUE FALSE  TRUE  TRUE
# 數字轉字串
as.character(a3)
[1] "-2" "-1" "0"  "1"  "2"
