排序 (sort vs. order)

  • sort: 將一數列由小排到大
  • order: 回傳數列個元素的大小順序
x <- c(1, 5, 3, 7)
sort(x)
[1] 1 3 5 7
order(x)
[1] 1 3 2 4
x[order(x)] # 等價於 sort(x)
[1] 1 3 5 7
sort(x, decreasing = TRUE) # 由大排到小
[1] 7 5 3 1

排序

order 可以處理多欄位的排序

df <- data.frame(x=c(1, 1, 3, 3), 
                 y=c(2, 4, 2, 4))
df[order(df$x, df$y),]
  x y
1 1 2
2 1 4
3 3 2
4 3 4
df[order(df$y, df$x),]
  x y
1 1 2
3 3 2
2 1 4
4 3 4

資料轉換 (Recoding Data)

  • rowSums：給定資料表，快速計算資料的列總和 (同理有colSums 計算欄總和)
  • 利用$colname 新增欄位
  • 學習cut函數將數值資料轉成類別資料
dat$total <- rowSums(dat[,7:10]) # read + write + math + science
dat$grade <- cut(dat$total,
  breaks = c(0, 140, 180, 210, 234, 300),
  labels = c("F", "D", "C", "B", "A"))
summary(dat[, c("total", "grade")])
